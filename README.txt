Java aplikacija koja simulira web prodavnicu oruzja. Aplikacija je napravljena pomocu JSP, Servleta i Hibernate-a. Postoje 3 nacina na koje korisnik moze pristupiti aplikaciji. Kao gost, bez naloga, korisnik moze samo da razgleda proizvode, bez mogucnosti kupovine. Kao administratior, korisnik moze ubacivati i brisati proizvode, kao i  menjati informacije postojecih proizvoda. Kao korisnik sa nalogom, moze obavljati kupovinu proizvoda. Projekat je radjen u NetBeans IDE-u.
